Documentação do Planilia

Guia completo de uso, funcionamento e dados técnicos da aplicação.

Visão Geral

Planilia é um portal para gestão pessoal: finanças, investimentos, lazer, imóveis e automóveis. Ele roda totalmente no navegador e usa localStorage para persistência.

Como Navegar

  • Início: visão geral e links úteis (documentação, tutorial).
  • Finanças: registre receitas e despesas, envie para atualizar o gráfico de evolução.
  • Investimentos / Lazer / Imóveis / Automóveis: formulários para cadastrar e visualizar itens.

Autenticação

O sistema salva usuários em localStorage na chave users. Você pode cadastrar múltiplos emails; todos podem fazer login enquanto mantiverem a mesma senha cadastrada. Existe também uma conta de teste: admin@klld.com / 123456.

Armazenamento (localStorage)

  • users: array de objetos { name, email, password }.
  • dadosFinanceiros: histórico de lançamentos [{ hora, saldo }].
  • minhaCarteira, meuLazer, meusImoveis, meusAutomoveis: arrays com itens salvos por seção.

Gráfico de Evolução

O gráfico usa Chart.js e tem filtros por período: Dia (últimas 24h), Semana (7 dias) e Mês (30 dias). Os botões na área financeira chamam funções que re-renderizam o gráfico usando os dados filtrados.

Formato de Datas

Para o filtro temporal funcionar corretamente, registros no histórico devem ter o campo hora com um destes formatos:

  1. DD/MM/YYYY HH:MM — data completa
  2. HH:MM — hora do dia (assume a data de hoje)

Como Resetar Dados

Para apagar todos os dados use o botão Limpar Tudo e Resetar Gráfico na seção de finanças, ou no console do navegador:

localStorage.clear();

Dicas e Problemas Comuns

  • Se o gráfico não aparece, verifique se existem registros em dadosFinanceiros.
  • Se não conseguir logar, confirme email e senha em users no console do navegador.
  • Em caso de caracteres estranhos ou problemas de encoding, tente recarregar a página e checar o arquivo script.js.
